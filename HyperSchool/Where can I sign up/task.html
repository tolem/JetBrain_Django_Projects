Description

Our site lacks one crucial page. This is the page where students enroll in courses. This is also where users submit their personal data to the app database.

Let's create a form for enrolling. Start with creating a required form in the forms.py file. Don't forget to list all the fields that are required to fill the Student model with data. There are two ways to do this â€” use ModelForm or Form and mention the required columns.

After that, create a view function in views.py for it. This view function works with the POST-method. It needs to check whether the form is valid. Don't forget to set up a URL to redirect users to the main and course_details page after enrolling.

Furthermore, add a new HTML template and URLs. It's necessary to create one more HTML template for this purpose. Set up the appropriate path to /schedule/add_course/ in urls.py. Place a link to the new page on the page with the course details.

Objectives
  • Create a new form for enrolling;
  • Create a new view function for it;
  • Add an HTML template and URLs.
