Description

Our site requires proper login and sign-up features. Django provides us with vast functionality on authentication and allows us to create groups and permissions. That'll be enough for us.

First, let's create HTML templates for login and sign-up by adding links to the sign-up and login pages at our main template. Please keep in mind that a link should redirect us to another page. Two new HTML templates in the templates' directory should contain the appropriate forms wrapped in tags <form>…</form>.

After this, add two new view functions for the templates in the views.py file. For the sign-up view function, import UserCreationForm. With this, you'll be able to create a user with no privileges — you'll require only a username and a password.

As usual, adjust the URL mapper. Each new feature with extra pages needs new paths in url.py.

Objectives
  • Create new HTML templates for login and sign-up;
  • Create view functions for the new templates;
  • Adjust the URL mapper.
